Unity C# Game Jam

Project Overview

制作期間2日間（学内ゲームジャム）
チーム構成チーム制作
使用技術Unity (C#)

学内ゲームジャムにて2日間で制作した、2人対戦型の弾幕シューティングゲームです。
短期間での開発において、インゲームの爽快感と、リトライ性の高いUI遷移の実装に注力しました。

Technical Highlights

01. UI演出とシーン遷移の実装

プレイヤーの操作感を損なわない、スムーズかつ直感的な画面遷移を実装しました。

  • 長押しゲージによる遷移制御:
    リザルト画面からタイトルやリトライへ進む際、単なるクリックではなく「長押し」による確定操作を採用。
    Image.fillAmount を動的に制御することで、ゲージが溜まる演出と遷移ロジックを連動させ、誤操作防止と没入感の向上を図りました。

02. 弾幕攻撃パターンの構築

シューティングゲームの核となる攻撃システムの実装を担当しました。

  • 360度弾幕攻撃:
    全方位に展開する弾幕攻撃のロジックを実装。対戦相手を追い詰めるための攻撃パターンを構築し、ゲームの競技性を高める要素として機能させました。
