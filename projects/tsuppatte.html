Unity C# Game Jam

Project Overview

制作期間2日間（ゲームジャム）
チーム構成チーム制作
使用技術Unity (C#)

この作品では、短期間開発におけるゲームバランスの設計と操作の気持ちよさを重視しました。
チーム開発ながら個々の役割を明確にし、プレイヤーが「もう一回やりたい」と思えるテンポを追求しました。

Technical Highlights

01. ゲームロジックの実装

プレイヤーの熱量をゲームに反映させるためのコアロジックを担当しました。

  • 入力回数と距離の変換:
    プレイヤーの連打入力を「距離」に変換する関数を設計。操作結果が即座に画面上の盛り上がりに反映されるよう調整しました。
  • バランス調整:
    短時間でもプレイヤーのボルテージが上がるようにパラメータ調整を行い、「競技性 × バカバカしさ」のちょうど良いラインを探りました。
