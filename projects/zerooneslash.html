Project Overview

使用技術Unity (C#), Firebase, DOTween, GitHub
受賞🏆 バンタン審査会 優勝（2026.02.17）

ダーツの要素を取り入れたスラッシュアクションゲームです。
ネオンレッドで統一されたビジュアルと、直感的なスラッシュ操作で「気持ちいい」体験を追求しました。
企画・設計・実装・UI/UXデザインまで全工程を一人で担当し、審査会で優勝を獲得しました。

Technical Highlights

01. Builderパターンによる動的UI生成

遊び方パネル、ランキング、リザルト画面などをBuilderパターンで設計しました。

  • [DefaultExecutionOrder(-5)] + Awake() でランタイム時に自動生成
  • HowToPlayBuilder, RankingPanelBuilder, NewRecordPanelBuilder, ResultPanelBuilder を実装
  • UIの追加・変更がコードの修正だけで完結する保守性の高い設計

02. ネオンレッドテーマによるUI/UX統一

ゲーム全体をネオンレッドのカラーテーマで統一し、没入感のあるビジュアルを実現しました。

  • カラーパレットを一元管理し、全UIに適用
  • 遊び方パネルは4ページスライド形式で、初見プレイヤーにも分かりやすい導線
  • DOTweenによるスムーズなアニメーション演出

03. Firebaseによるオンラインランキング

Firebase Realtime Databaseを使用し、リアルタイムのスコアランキング機能を実装しました。

  • スコア送信・取得の非同期処理
  • ニューレコード検知とパネル表示の連携
